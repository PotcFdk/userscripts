# userscripts
A collection of small/simple user scripts.

### [BackToOldReddit](BackToOldReddit)
Ensures that you stay on old.reddit.com.
### [HeiseThreadEnabler](HeiseThreadEnabler)
Auto-expands the thread view in the forums / comment sections.
### [SteamLinkFilterBypass](SteamLinkFilterBypass)
Skips the Steam link filter.
### [ZenMarketHistoryTitles](ZenMarketHistoryTitles)
Adds buttons to make the Auction Watchlist History more readable by replacing Item IDs with Item Titles.
